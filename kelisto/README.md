Test for Kelisto 

# Type of test
Shopping Cart

# Technologie used
React 13.16.1 \

# How to install

* `git clone` this repo
* `npm install`
* run `npm start`
* open browser on `http://localhost:4200/`

* If you prefer to run in a different host, please run:
* npm start -- --port [host]
